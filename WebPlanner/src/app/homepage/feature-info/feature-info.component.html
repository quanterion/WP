Features & Benefits

Truly online application

  • Works in any modern browser without extra plugins, thanks to HTML5 & WebGL
  • Designed to work on any internet-connected device. Can be used on PCs, tablets, and mobile phones
  • Can be run on Windows, Linux, Mac, Android, and iPhone.. anywhere!
  • Supports realtime collaboration. Any changes to projects and catalogs are immediately visible to all application users

Integrated catalog manager

  • Import your models from multitude of modeling software files (KOMPAS(c3d), 3DS, OBJ, DAE, STL, STEP, IGES, VRML(wrl), Parasolid(x_t, x_b), ACIS(sat), JT models, Blender(blend))
  • Import custom windows & doors as simple as any other models
  • Group models in folders
  • Easy texture covering using drag&drop with automatic conversion to suitable size
  • You can configure multiple variants of the same model with different materials and prices using properties
  • Support animation in models that can be used for doors, drawers, sofas, etc

Powerful floor plan editor

  • Interactive WYSIWYG editor
  • Great support for multiroom appartment modeling
  • Works consistently on 2D and 3D views
  • Smart and exact editing of any dimension
  • Import custom doors and windows from any 3D format
  • Models real-world appartments with non-orthogonal walls and different wall thickness
  • Supports multistorey house modeling

High quality WebGL visualization

  • High quality shadows and reflections in realtime
  • Flexible material configuration
  • Drag&drop textures from local computer
  • Supports latest WebGL2 in modern browsers
  • Various shaded and wireframe visualization styles
  • Hyperlink generation from camera position

Furniture placement

  • Intuitive drag&drop of models
  • Collision detection prevents penetration of models into each other
  • Supports smart model resizing
  • Display accurate distances to nearby objects taking into account room shape
